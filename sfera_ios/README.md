# Sfera — клиент для iPhone

Нативное iOS-приложение (Swift + WKWebView) для мессенджера **Sfera**.  
Подключается к вашему серверу по сети — тот же интерфейс, что в браузере на телефоне.

**Сервер по умолчанию:** `http://212.220.113.9:10124`

> **С Windows:** Xcode на Windows не ставится, но IPA можно собрать через **облачный Mac** (Codemagic, GitHub Actions) или вообще без сборки — **PWA в Safari**.  
> Подробно: **[BUILD_ON_WINDOWS.md](BUILD_ON_WINDOWS.md)**

---

## Как установить на iPhone бесплатно (выберите способ)

### Способ 1 — PWA без Mac (самый быстрый, ~30 секунд)

Подходит для тестов, если не нужен отдельный `.ipa`. Работает как иконка на домашнем экране.

1. На iPhone откройте **Safari**
2. Перейдите на `http://212.220.113.9:10124`
3. Нажмите **Поделиться** (квадрат со стрелкой вверх)
4. **«На экран "Домой"»** → **Добавить**
5. Готово — иконка **Sfera** на рабочем столе

Сервер должен быть запущен, порт **10124** открыт в файрволе.

> Минус: нет иконки в App Store, Safari иногда перезагружает вкладку. Для полноценного приложения — способ 2.

---

### Способ 2 — Xcode + бесплатный Apple ID (нужен Mac, ~10 минут)

Apple позволяет ставить **свои** приложения на **свой** iPhone **бесплатно** (Personal Team).  
Приложение работает **7 дней**, потом снова нажать ▶ Run в Xcode (или переустановить).

#### Что нужно

- Mac с **Xcode** (из App Store, бесплатно)
- iPhone + кабель USB (или Wi‑Fi debugging)
- Бесплатный Apple ID (ваш iCloud)

#### Шаги

1. **Установите Xcode** из App Store на Mac
2. Скопируйте папку `sfera_ios` на Mac (флешка, iCloud, Git)
3. Откройте **`Sfera.xcodeproj`** двойным щелчком
4. В Xcode: **Xcode → Settings → Accounts** → **+** → войдите Apple ID
5. В проекте слева выберите **Sfera** (синяя иконка) → вкладка **Signing & Capabilities**
   - Включите **Automatically manage signing**
   - **Team** → ваш Apple ID (Personal Team)
   - Если Bundle ID занят — измените `com.sfera.messenger` на уникальный, например `com.вашеимя.sfera`
6. Подключите iPhone, разблокируйте, на телефоне нажмите **«Доверять»** этому компьютеру
7. Вверху Xcode выберите ваш **iPhone** (не Simulator)
8. Нажмите **▶ Run** (Cmd+R)
9. На iPhone: **Настройки → Основные → VPN и управление устройством** → доверьте разработчику (вашему Apple ID)

Готово. Приложение **Sfera** на домашнем экране.

#### Смена адреса сервера

В приложении: **⚙** (шестерёнка) → укажите URL, например `http://212.220.113.9:10124`

#### Через 7 дней

Приложение перестанет открываться — снова подключите iPhone к Mac и нажмите **▶ Run** в Xcode (переустановка занимает ~1 минуту).

### Способ 3 — Codemagic + Sideloadly (Windows, без своего Mac)

1. Зарегистрируйтесь на [codemagic.io](https://codemagic.io)
2. Подключите репозиторий с папкой `sfera_ios` (или загрузите zip)
3. Привяжите **Apple ID** в настройках Codemagic
4. Запустите сборку → скачайте **.ipa**
5. На Windows: [Sideloadly](https://sideloadly.io) → установите IPA на iPhone

Пошагово: **[BUILD_ON_WINDOWS.md](BUILD_ON_WINDOWS.md)**

---

### Способ 4 — AltStore / Sideloadly (если есть готовый .ipa)

Если кто-то собрал `.ipa` на Mac, можно установить с Windows через [Sideloadly](https://sideloadly.io) или AltStore — нужен Apple ID.  
Срок подписи — **7 дней** (бесплатный аккаунт).  
Сборка `.ipa` без Mac в этом проекте не описана — используйте способ 1 или 2.

## Сравнение способов

| Способ | Mac | Windows | Стоимость | Время |
|--------|-----|---------|-----------|-------|
| PWA (Safari) | Нет | Да | Бесплатно | ~30 сек |
| Codemagic + Sideloadly | Нет* | Да | Бесплатный тариф | ~15 мин |
| Xcode + Apple ID | Да | Нет | Бесплатно | ~10 мин |
| GitHub Actions | Нет* | Да (git push) | Бесплатно** | ~10 мин |

\* Сборка на **облачном Mac**, не на вашем ПК  
\*\* Публичный репозиторий — бесплатные минуты macOS runner

## Связь с основным проектом

Клиент **не копирует** файлы из `discord_clone/client` — он загружает UI с сервера.  
Обновления интерфейса на сервере видны сразу, без пересборки приложения.

## Частые проблемы

**«Не удаётся подключиться»** — проверьте URL (`http://...:10124`), Wi‑Fi/мобильный интернет, файрвол на сервере.

**Xcode: Signing failed** — выберите Team (Apple ID), смените Bundle Identifier на уникальный.

**На iPhone: «Ненадёжный разработчик»** — Настройки → Основные → VPN и управление устройством → Доверять.

**Голос не работает** — разрешите микрофон для Sfera в Настройки → Конфиденциальность.
