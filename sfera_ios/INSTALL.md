# Установка на iPhone

## С Windows (без Mac) — самое быстрое

**PWA (~30 сек):** Safari → `http://212.220.113.9:10124` → Поделиться → На экран «Домой»

**Нативное IPA:** Codemagic (облако) → Sideloadly на Windows → см. **[BUILD_ON_WINDOWS.md](BUILD_ON_WINDOWS.md)**

---

## С Mac (бесплатный Apple ID)

1. Установить **Xcode** из App Store
2. Открыть **`Sfera.xcodeproj`**
3. **Signing & Capabilities** → Team = ваш Apple ID
4. Подключить iPhone → **▶ Run**

Подробности: [README.md](README.md)
